# Dashboard de Vendas – Análise Anual e Performance por Categoria

Este projeto apresenta um **dashboard de vendas interativo**, desenvolvido para oferecer uma visão clara e estratégica do **resultado anual de vendas de uma empresa**.

O foco do dashboard é facilitar análises rápidas e apoiar na tomada de decisão por áreas como comercial e marketing.

## Funcionalidades do Dashboard

### ➡ Indicadores Principais (KPIs)
O dashboard apresenta os principais indicadores de desempenho de vendas:
- **Faturamento Total**
- **Total de Vendas**
- **Ticket Médio**
- **Clientes Atendidos**

Esses KPIs são atualizados dinamicamente conforme os filtros aplicados.

### ➡ Análise Dinâmica por Categoria
O usuário pode selecionar diferentes categorias de produtos, como: notebook, tablet, relógios...

A partir dessa seleção, o dashboard se adapta automaticamente, atualizando:
- Os KPIs
- Os gráficos
- A análise de faturamento por marca.

### ➡ Faturamento Total por Marca
O dashboard também apresenta a **distribuição do faturamento por marca**, possibilitando identificar as marcas mais relevantes para o faturamento anual

---

## ➡ Benefícios para o Negócio

Este dashboard permite:
- Acompanhar a **performance anual de vendas**
- Identificar **categorias e marcas mais rentáveis**
- Ajuste de estratégias comerciais
- Foco em produtos com maior retorno
- Otimização do portfólio de marcas
